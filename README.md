# 📊-Telecom-Churn-Analysis-Prediction

## 📌 Overview

This project focuses on analyzing telecom customer data to understand **customer churn behavior** and predict future churners.
It covers a complete end-to-end pipeline including **data extraction, transformation, visualization, and machine learning prediction**.

The main objectives are:

* Analyze customer data across multiple dimensions
* Identify patterns and reasons behind churn
* Predict customers who are likely to churn in the future

---

## 📊 Dashboard Explanation

### 🔹 1. Executive Summary Page

This page provides an overview of historical data and key metrics:

* **Total Customers, New Joiners, Total Churn, Churn Rate**
* Customer segmentation based on:

  * **Demographics** → Gender & Age group analysis
  * **Account Info** → Payment method, contract type, tenure
  * **Geography** → Churn rate by state
  * **Services Used** → Impact of services on churn
  * **Churn Distribution** → Reasons for customer churn

---

### 🔹 2. Churn Prediction Page

This page highlights **future churners using a Machine Learning model**:

* Displays **predicted churn customers**
* Shows breakdown by:

  * Gender, Age, Marital Status
  * Tenure, Payment Method, Contract
  * State-wise churn distribution
* Includes a detailed table with:

  * Customer ID
  * Revenue metrics
  * Referrals & charges

---

## 🛠️ Tools & Technologies

* **SQL Server (SSMS)** → Data extraction, cleaning, and transformation (ETL)
* **Power BI** → Data modeling, DAX measures, dashboard creation
* **Python (Jupyter Notebook)** → Machine learning model building
* **Machine Learning** → Random Forest Classifier for churn prediction

---

## 🔍 Key Insights

* 📉 Overall churn rate is significant, indicating customer retention issues
* 👩‍🦳 Customers aged **50+ (especially females)** show higher churn
* 💳 Customers with **monthly contracts** churn more than long-term contracts
* 🧾 Certain services (like lack of support services) increase churn risk
* 🤖 ML model achieved **~84% accuracy** and identified **378 high-risk customers**

---

## 🚀 Features

* Interactive dashboard with **cross-filtering**
* Drill-down capability for detailed analysis
* Slicers for dynamic filtering
* Customer-level churn insights
* AI-based churn prediction

---

## ▶️ How to Use

1. Open the Power BI dashboard file
2. Use filters (top slicers) to explore specific customer segments
3. Click on visuals to apply cross-filtering
4. Navigate between:

   * **Summary Page** → Historical analysis
   * **Prediction Page** → Future churn insights
5. Analyze individual customers from the prediction table
